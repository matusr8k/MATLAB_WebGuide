最初に

このページは、MATLABを用いて研究データを解析し、整理されたデータグラフを得ることをサポートする目的で作られているWebガイドです。
MATLABドキュメント内の必要な情報を簡素にまとめてガイドを作成しているため、より詳細で正確な情報はMATLABの公式ドキュメントをご確認ください。

サンプルコードの使い方

ガイドを分かりやすく活用できるようにM-ファイルで記述されたサンプルコードを用意しました。
以下のリンクからGithubに移動して、緑色のClone or download＞Desktop ZIPよりダウンロードが可能です。

  • GITHUB SAMPLE CODE

ダウンロードしたフォルダのMATLABサンプルコードより各ガイドに対応したM-ファイルを選択し、MATLABで開きます。
開いたら、MATLABの画面左上のタブのエディタを選択し、実行からサンプルコードを呼び出すことができます。

ガイドを編集される方へ

ガイドの編集を行うには、まずGithubにサインインする必要があります。アカウントを持っていない場合は、Sign Upからアカウントを登録することができます。

アカウントを用意したら、上記のGITHUB SAMPLE CODEからGithubのサンプルコードのあるページへ移動して、画面右上にあるForkボタンを押します。

Forkすると、左上のmatusr8k/MATLAB_WebGuideがあなたのusername/MATLAB_WebGuideに変わります。そして、緑色のボタンからcloneを行います。

既存のガイドを編集する場合は、guides/対象のhtmlに変更を加えます。その際にガイドのヘッダーに更新日、Githubのユーザー名、対象となるMATLABのバージョンを記述していただければ幸いです。
そして変更点を表記してプルリクエストを行います。管理者が確認を行った後、変更点がサイトに反映されます。

新規のガイドを作成する場合は、guides/任意のガイド.htmlを複製し、名前と内容を書き換えます。そして新規のガイドであることを表記してプルリクエストを行います。管理者が確認を行った後に記事がメニューに加えられ、サイトに反映されます。
