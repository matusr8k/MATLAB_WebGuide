多項式の定義

不定元x(1変数)の多項式は行ベクトルによって表され、例えば4x² + 2x + 1を表す場合は

fx = [4 2 1];

のように表記します。

ただし、5x³ - 2xのような多項式を扱う場合、空いている項には0を明示する必要があります。

fx = [5 0 -2 0];

多項式の演算

多項式fxの計算を行う場合はpolyval(fx, x)を使用します。xには変数の値を入力します。
また、多項式fxの解を求める場合はroots(fx)を使用します。解はn次多項式の場合、n×1の列ベクトルを返します。

また、多項式fxを微分積分することも可能です。微分するにはpolyder(fx)を、積分するにはpolyint(fx)を使用します。
